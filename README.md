 Student Lead Management App (Mini CRM)

This project is a simple React Native application built as part of an internship screening assignment.  
The goal of the app is to manage student leads in a clean and easy-to-use way.

Features
- Add new student leads using a form  
- Display all leads in a list using FlatList  
- View complete details of a selected lead  
- Delete existing leads  
- Navigate smoothly between screens using React Navigation  
- Basic form validation  
- Data persistence using AsyncStorage (leads remain after app reload)

 Tech Stack
- React Native (Expo)
- React Hooks
- React Navigation
- AsyncStorage

 Screens in the App
1. Lead List Screen  
2. Add Lead Screen  
3. Lead Details Screen  

How to Run the Project
1. Run `npm install`  
2. Start the project using `npx expo start`  
3. Open the app in browser or scan the QR code with Expo Go
